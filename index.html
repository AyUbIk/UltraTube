Дизайн уровня топовых платформ. Поддержка телефонов и десктопов. Отправляй ссылки прямо в бота — быстро и красиво.

Отправьте ссылку на любое видео и админы канала посмторят его и снимут реакцию
Готово к публикации: GitHub Pages / Netlify / Vercel
Автор — Админы канала

Режим "быстро" отправляет напрямую на Telegram API (открывает токен — небезопасно).

Инструкции:
  1. Рекомендуется: развернуть серверless-функцию (пример внизу) и указать её URL в ultratube.js → CONFIG.CLIENT_API.
  2. Если понимаете риски — можете временно вставить BOT_TOKEN и CHAT_ID в конфиг (ultratube.js), но на продакшене это опасно.
  3. Опубликуйте сайт на GitHub Pages / Netlify / Vercel и прокси на Vercel/Netlify с переменными окружения BOT_TOKEN и CHAT_ID.
